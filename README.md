# CMSC-207 Mini-Project
This project is my personal web portfolio created for CMSC 207. It is inspired by the book Create with Code: Build Your Own Website by Clyde Hatter.

This website introduces who I am, my interests, and how to contact me. It is built using basic web development technologies:

* HTML for the structure
* CSS for design and layout
* JavaScript for simple interactivity

The goal of this project is to practice creating a clean, responsive, and well-structured personal website. It serves as a simple portfolio that showcases my foundational skills in web development.
